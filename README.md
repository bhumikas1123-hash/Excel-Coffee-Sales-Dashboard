# Coffee Sales Dashboard ☕ 

## Project Overview

This project presents an interactive Coffee Sales Dashboard created to analyze sales performance, customer behavior, and product trends using Excel. The dashboard transforms raw transactional data into meaningful business insights through data cleaning, analysis, and visualization techniques.

The goal of this project is to help understand:

* Overall sales performance
* Top-performing coffee products
* Customer purchasing patterns
* Regional sales distribution
* Monthly and yearly revenue trends

# Dataset Information

The dataset contains coffee sales transaction records including:

* Order Date
* Customer Name
* Product Type
* Coffee Size
* Quantity Sold
* Sales Amount
* Country/Region
* Customer Details

The data was cleaned and transformed before building the dashboard to ensure accurate reporting and analysis.

---

#  Key Dashboard Features

### ✔ Sales Performance Analysis

Tracks total sales, revenue growth, and order trends over time.

### ✔ Product Analysis

Identifies top-selling coffee products and categories contributing most to revenue.

### ✔ Customer Insights

Analyzes customer purchasing behavior and repeat purchases.

### ✔ Regional Analysis

Visualizes sales performance across different countries/regions.

### ✔ Interactive Filtering

Slicers allow users to dynamically filter data by:

* Time Period
* Product Type
* Roast Type
* Size
* Country

---

# Key Insights

* Arabica and Excelsa coffee types generated strong sales performance.
* Certain months showed higher revenue trends indicating seasonal demand.
* Larger coffee sizes contributed significantly to total revenue.
* Some regions consistently outperformed others in overall sales.
* Repeat customers played an important role in revenue generation.

5. Explore insights and trends from the visualizations.
